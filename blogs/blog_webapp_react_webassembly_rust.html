Create a Web App using React, Redux, WebAssembly (Rust), and webpack

Follow @cosmoplankton

In this tutorial, we would go through the steps to create a web app using:

  • WebAssembly compiled from Rust; to perform computation and draw on the canvas.
  • React; to build an overlay user interface on top of the canvas.
  • Redux; to provide the store for the app.
  • Cargo, wasm-pack, wasm-bindgen, yarn, and webpack as tools in our toolchain.

Here, we will discuss the setup of the development environment and the bootstrap code for the different modules. We won't go into implementation details (maybe in a future article), but the complete code is available at this repository for reference.

Environment setup

We will use Windows Subsystem For Linux (wsl2) with Visual Studio Code (vs-code) for the development environment and IDE. This way the steps and commands would be valid irrespective of the OS - Linux or Windows. For steps on how to set up wsl and vs-code check this earlier article.

# start wsl from Windows Terminal
wsl
# create a development directory and launch VSCode
mkdir wasm-react-redux-webapp && $_
code .

Setup git, Rust, nvm, and yarn:

git init
git commit --allow-empty -m "empty init"
git branch -m main
git checkout -b dev
touch README.md
# install rustc and cargo using rustup
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# setup node and nvm
sudo apt update && sudo apt upgrade
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash
command -v nvm
nvm ls-remote
nvm install --lts
nvm ls
nvm use --lts
npm --version
node --version
npm install -g yarn
yarn set version latest
yarn init

Edit webpack.common.js, webpack.dev.js, and webpack.prod.js.

webpack.common.js config file:

...
const wasmPackPlugin = require('@wasm-tool/wasm-pack-plugin');
const HtmlWebpackPlugin = require('html-webpack-plugin');
const { CleanWebpackPlugin } = require('clean-webpack-plugin');
const CopyWebpackPlugin = require('copy-webpack-plugin');

let pages = ["index", "webgl", "webgpu"]
let plugins = [
    new CleanWebpackPlugin(),
    new CopyWebpackPlugin({
        patterns: [
            { from: 'static' }
        ]
    }),
    new wasmPackPlugin({
        crateDirectory: path.resolve(__dirname, "app-core"),
        outDir: path.resolve(__dirname, "app-core/pkg"),
        outName: "app-core",
    })
];
plugins = plugins.concat(
    pages.map(
        (page) => 
            new HtmlWebpackPlugin({
                template: path.resolve(__dirname, `src/pages/${page}-template.html`),
                filename: `${page}.html`,
                chunks: [page],
              })
      )
)

module.exports = {
    mode: 'production',
    entry: pages.reduce((config, page) => {
        config[page] = path.resolve(__dirname, `src/pages/${page}.js`);
        return config;
      }, {}),
    output: {
        filename: '[name].bundle.js',
        path: path.resolve(__dirname, 'dist'),
        devtoolNamespace: 'devtool-namespace'
    },
    plugins: plugins,
    experiments: {
        syncWebAssembly: true,
    },
    module: {
        rules: [
            ...
        ]
    },
    ...
};

Clone and Run

To run the app, follow the below steps:

# Make sure you have set up the environment (nvm, Cargo, etc.)
# git-URL: https://github.com/cosmoplankton-studio/expt-src-cosmoplankton.github.io.git
git clone git-URL wasm-react-redux-webapp # replace the URL 
cd wasm-react-redux-webapp
git checkout dev
yarn install
# upgrade packages if necessary
yarn upgrade [package]@[version] # optional
# build and serve on localhost
yarn build
yarn serve
